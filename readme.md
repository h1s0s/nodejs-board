# Node.js Restful API 게시판 프로젝트

Node.js 기반 Restful API 게시판 개발 프로젝트 입니다

### 기술 스택
Node.js(express), MySQL, sequelize, swagger ui

### 주요 기능
1. 게시물 CRUD API 개발
2. Ajax로 비동기 처리
3. 시퀄라이즈(sequelize) 라이브러리를 사용하여 DB 통신
